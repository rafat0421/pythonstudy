# Week 2 — Conditions and Decision Logic Complete Submission

This folder contains all required Week 2 task parts plus the optional challenge.

## Files

- `part1_week2_conditions.py` — completed Part 1 grade classifier and login validator.
- `part1_sample_run.md` — example test runs for Part 1.
- `part2_decision_logic_explanation.md` — answers to all Part 2 questions.
- `part3_debugging_answer.md` — answers to all Part 3 debugging questions.
- `part4_reflection_answers.md` — answers to all Part 4 reflection questions.
- `part5_optional_challenge.py` — optional challenge version.
- `chatgpt_log.md` — sample task-related ChatGPT interaction log.
- `submission_values.md` — confidence, dependency, and time used.

## How to run Part 1

```bash
python part1_week2_conditions.py
```
