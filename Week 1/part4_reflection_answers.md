# Part 4 — Reflection on ChatGPT Use

## 1. What did you learn this week?

This week I learned how to combine variables, user input, data conversion, arithmetic operations, and conditional logic in one small Python program. The most important lesson was that `input()` returns text, so I need to convert user input to a numeric type before doing calculations. I also learned why division by zero needs separate handling.

## 2. What part was most difficult?

The most difficult part was thinking through the error cases instead of only focusing on the normal calculation. At first, I concentrated on making addition work, but I then needed to consider what happens when the user chooses an unsupported operation or tries to divide by zero.

## 3. How did you use ChatGPT?

I used ChatGPT mainly for explanations and verification. I asked about the difference between strings and numbers, how `float()` works, and where the division-by-zero check should be placed. I also used it to review whether my conditional structure covered all required operations.

## 4. Did ChatGPT mainly explain, guide, verify, debug, or solve?

ChatGPT mainly explained and verified. It helped me understand the purpose of `float()` and the structure of `if`, `elif`, and `else`. It also guided me on how to think about the division-by-zero case. I used the explanations to understand and revise the code rather than treating the response as something to submit without review.

## 5. Could you solve a similar task without ChatGPT?

I think I could solve a similar basic calculator task without ChatGPT now. I understand the overall flow: collect input, convert numbers, select an operation with conditions, and handle invalid cases. I might still need to check syntax for a more advanced version with functions or exception handling, but I could build the basic version myself.

## 6. Confidence level, 1–5

**4 / 5**

I feel fairly confident with the main concepts from this task. I understand the logic, but I need more practice to write similar programs quickly without checking examples.

## 7. Dependency level, 1–5

**3 / 5**

I used ChatGPT as a learning aid, especially for explanations and verification. I was able to understand the final solution, but I still relied on support when I was uncertain about error handling.

## 8. What would you do differently next time?

Next time, I would first write a short plan before coding. I would list the inputs, possible operations, expected output, and error cases. I would also test each operation one by one before considering the task complete. Before asking ChatGPT, I would try to identify the problem myself and write down what I think is happening.
