# Week 1 — Python Foundations and GPT Learning Habits

This folder contains a complete Week 1 submission package for the thesis study.

## Files

- `week1_calculator.py` — Required calculator task.
- `week1_optional_challenge.py` — Optional challenge: functions, repeated calculations, input validation, and history.
- `part2_code_explanation.md` — Answers to every required code-explanation question.
- `part3_debugging_answer.md` — Answers to every debugging question.
- `part4_reflection_answers.md` — Reflection answers for the study platform.
- `chatgpt_log.md` — Study-related AI-use record.
- `submission_details.md` — Suggested values for the upload form.

## Before submission

Review `part4_reflection_answers.md`, `chatgpt_log.md`, and `submission_details.md` and make sure that the first-person statements, ratings, and time used accurately reflect your own experience.
